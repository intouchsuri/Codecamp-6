ให้หาว่า input type มีอะไรบ้างแต่ละ type แตกต่างกันอย่างไร

Type value
button ปุ่มกด
checkbox เลือกติ๋กถูก
color เลือกแสดงสี
date รับวันที่โดยค่าเริ่มต้นเราเป็นผู้กำหนด
date-local รับวันที่โดยค่าเริ่มต้นเราเป็นผู้กำหนด เมื่อเลือกจะทำการเอาวันที่เครื่องมาแสดง
e-mail รับค่าเป็น e-mail ตรวจสอบว่ามี @
file upload file
hidden ไม่แสดงข้อความที่หน้าจอ
img upload ไฟล์รูปภาพเท่านั้น
month รับข้อมูลเฉพาะเดือนปี
number รับข้อมูลเป็นตัวเลขเท่านั้น
password ขณะรับข้อมูลจะแสดงผลเป็นจุดกลมๆเท่านั้น
radio รับข้อมูลเป็น radio box โดยเลือกได้เพียง 1 รายการเท่านั้น
range รับข้อมูลเป็นระดับสไลด์บนจอด้วย mouse ได้
reset ปุ่มยกเลิกการกรอกข้อมูลภายในฟอร์มเดียวกัน
search กล่องสำหรับ ค้นหา
submit ปุ่มยืนยัดส่งค่าต่างๆ
tel ใส่หมายเลขโทรศัพท์
text ใส่ข้อความได้ยาวๆ
time รับข้อมูลเป็นเวลา
url รับข้อมูบเป็นประเภท webside
week รับข้อมูลเป็นสัปดาห์และปีที่เลือก
